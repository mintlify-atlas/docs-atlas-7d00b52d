## Inicio Rápido

<Steps>
  <Step title="Clona el repositorio">
    Descarga el código fuente desde GitHub.
    ```bash
    git clone https://github.com/marvinalban30/CSS3-avanzado-y-Bootstrap-5.git
    cd "CSS3-avanzado-y-Bootstrap-5/CSS3 avanzado y Bootstrap 5"
    ```
  </Step>
  <Step title="Abre el sitio en tu navegador">
    No se requiere servidor ni herramienta de compilación. Abre `index.html` directamente.
    ```bash
    open index.html
    # En Linux:
    xdg-open index.html
    # En Windows:
    start index.html
    ```
  </Step>
  <Step title="Personaliza los estilos">
    Edita `styles.css` para modificar colores, tipografía y efectos visuales según tus necesidades.
  </Step>
  <Step title="Despliega el sitio">
    Publica el contenido en cualquier hosting estático: GitHub Pages, Netlify o Vercel sin configuración adicional.
  </Step>
</Steps>
